What is the document object model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, allowing scripts (like JavaScript) to manipulate and interact with the content, structure, and style of a web page dynamically.

How do you select an element from DOM?

In JavaScript, there are several methods to select elements from the DOM: 1. getElementById: Selects an element by its unique ID attribute. Example: document.getElementById("myElement"); 2. getElementsByClassName: Selects all elements with a specific class name. Returns an HTMLCollection. Example: document.getElementsByClassName("myClass"); 3. getElementsByTagName: Selects all elements with a specific tag name. Returns an HTMLCollection. Example: document.getElementsByTagName("p"); 4. querySelector: Selects the first element that matches a CSS selector. Example: document.querySelector(".myClass"); 5. querySelectorAll: Selects all elements that match a CSS selector. Returns a NodeList. Example: document.querySelectorAll("p.myClass");

What is event delegation in the context of the DOM, and why it is useful?

Event delegation is a technique in which a single event handler is attached to a parent element, and the event is propagated to its child elements. This avoids the need to attach event handlers to individual child elements, which can be inefficient for large numbers of elements.

How do you manipulate an elements attribute and styles using the DOM?

Attributes: Get attribute: element.getAttribute("attributeName"); Set attribute: element.setAttribute("attributeName", "value"); Remove attribute: element.removeAttribute("attributeName"); Example: JavaScript var link = document.getElementById("myLink"); link.getAttribute("href"); // Get the href attribute link.setAttribute("target", "_blank"); // Set the target attribute link.removeAttribute("rel"); // Remove the rel attribute Use code with caution. Styles: Get style: element.style.propertyName; Set style: element.style.propertyName = "value"; Example: JavaScript var paragraph = document.getElementById("myParagraph"); paragraph.style.color = "red"; paragraph.style.fontSize = "20px";
